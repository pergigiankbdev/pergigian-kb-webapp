# Pergigian Kelantan Web App 🦷

A web-based application designed to streamline and manage dental health services, appointments, or information specifically tailored for the Kelantan region. Built using Python, this project leverages modern web development practices to deliver a reliable healthcare solution.

---

## 🚀 Features

*   **User Dashboard:** Intuitive interface for patients and clinic administrators.
*   **Appointment Management:** Easily schedule, reschedule, or cancel dental appointments.
*   **Clinic Locator:** Find participating dental clinics across Kelantan.
*   **Bilingual Support:** Designed with local accessibility in mind (Bahasa Melayu & English).

---

## 🛠️ Tech Stack

*   **Backend:** Python (Flask / Django)
*   **Frontend:** HTML5, CSS3, JavaScript (Bootstrap/Tailwind)
*   **Version Control:** Git & GitHub
